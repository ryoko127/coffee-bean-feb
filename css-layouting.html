Position

position: static; - this is the default positioning;
the element will stick to the normal page flow.
So if there is a left/right/top/bottom/z-index set then there will be no effect on that element.
position: relative; - is an element original position
the element will stick to the normal page flow. just like the static value.But now left/right/top/bottom/z-index will work. The positional properties can move the element from the original position in that direction.
position:absolute; - the element will be removed from the flow of the page abd other element will behave as if its not even there. all other positional properties will work on it
the element is removed from the flow of the page. relative to the document, not any particular parent. not affected by scrolling.
